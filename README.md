# Как запустить сайт (бесплатно/почти бесплатно, шаг за шагом)

Тебе понадобится 3 аккаунта: DeepSeek (ключ нейросети), GitHub (хранилище кода), Vercel (хостинг сайта). Все три регистрируются бесплатно, карта не нужна.

**Важно про DeepSeek:** на новый аккаунт дают 5 миллионов бесплатных токенов. После того как они закончатся — оплата по факту использования, но цена очень низкая (доли цента за обработку одной строки), так что для загрузки Excel "по необходимости" расходы будут минимальными.

## Шаг 1. Получи ключ DeepSeek API

1. Зайди на https://platform.deepseek.com и зарегистрируйся (email или телефон).
2. В личном кабинете найди раздел "API Keys" (обычно в левом меню).
3. Нажми "Create new API key", дай ему любое имя, скопируй ключ (строка вида `sk-...`) — он показывается только один раз, сохрани его сразу.

## Шаг 2. Залей код на GitHub

1. Зайди на https://github.com и зарегистрируйся (если нет аккаунта).
2. Нажми "New repository", назови например `spareparts-translator`, сделай Public, нажми "Create repository".
3. На странице нового репозитория нажми "uploading an existing file".
4. Перетащи туда ВСЕ файлы из папки `spareparts-translator`, которую я тебе передал (index.html, vercel.json, package.json, и папку api с файлом translate.js внутри).
5. Нажми "Commit changes".

## Шаг 3. Разверни сайт на Vercel

1. Зайди на https://vercel.com и зарегистрируйся через GitHub-аккаунт (кнопка "Continue with GitHub").
2. Нажми "Add New..." → "Project".
3. Найди свой репозиторий `spareparts-translator` и нажми "Import".
4. Перед нажатием "Deploy" разверни раздел "Environment Variables" и добавь:
   - Name: `DEEPSEEK_API_KEY`
   - Value: (ключ, который ты скопировал в шаге 1)
5. Нажми "Deploy" и подожди 30-60 секунд.
6. Vercel выдаст тебе ссылку вида `https://spareparts-translator-xxxx.vercel.app` — это и есть твой сайт.

## Шаг 4. Пользуйся

1. Открой ссылку из шага 3.
2. Загрузи Excel-файл со списком запчастей.
3. Выбери лист и колонку с названиями.
4. Нажми "Перевести и сгенерировать файл".
5. Файл с двумя новыми колонками скачается автоматически.

## Если что-то не работает

- Проверь, что ключ `DEEPSEEK_API_KEY` правильно вставлен в Vercel (Settings → Environment Variables проекта).
- Если меняешь код — после загрузки изменений на GitHub Vercel сам пересоберёт сайт за ~30 секунд.
- Если бесплатные токены закончились — пополни баланс на platform.deepseek.com (минимальные суммы, расходы для твоих объёмов будут небольшими).
